This plug in tells zodiac sign of the current month.

== Description ==

The purpose to design this plug in is to provide zodiac sign of the current month. 


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/current-zodiac` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use short code in post [current-zodiac].
